# 小说网站选择

首先, 做小说网必须获取资源, 资源靠爬虫抓取其他的小说网站, 
被抓的网站则是人工选择的. 网站一般选择非主流站点和无需登录的网站.

对于贴吧等资源就不做抓取了. 也不做特定目标抓取, 也就是不根据小说名抓取.
而是网站有什么就抓取什么.

不选取什么:
* 正版小说网
* 需登录
* 贴吧等集散资源
* 不按小说名全网搜索抓取
* 最好不需要https
